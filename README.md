# Alphard

## Prerequisites

* [Python 3.10+](https://www.python.org/)

## Setup (Windows)

1. Create and activate a virtual environment

   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```

2. Upgrade `pip`

   ```bash
   python -m pip install --upgrade pip
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```
